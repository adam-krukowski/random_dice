Installation
Clone the repository:

git clone https://github.com/adam-krukowski/random_dice.git

Navigate to the project directory:

cd random_dice


Open index.html in a web browser:
You can simply double-click the index.html file or open it with your favorite browser.

Project Structure
show.html.erb: The main HTML file containing the structure of the game.
application.css: The CSS file for styling the dice, buttons, and other elements.
index.js: The JavaScript file containing the game logic for rolling the dice and updating the UI.


Customization
You can modify the following aspects to customize the game:

Dice appearance: Update the CSS styles in style.css to change the size, color, or animation effects.
Dot positions: Adjust the dot layout in the JavaScript file if you want to customize the pattern for each dice roll.
